# Student Authentication System

This is a Student Authentication System developed using PHP and MySQL as part of my Web Development Internship at ApexPlanet Software Pvt. Ltd.

The project allows users to register and log in securely. Based on the user role, different features are available. Administrators can add, edit, delete, and view student records, while students can only view the records.

## Features

- User Registration and Login
- Admin and Student Role Management
- Add, View, Edit, and Delete Student Records
- Search Students by Name or Email
- Pagination
- Prepared Statements
- Client-side Validation
- Server-side Validation
- Session Management
- Logout and Delete Confirmation

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Project Modules

- User Authentication
- Dashboard
- Student Management
- Search and Pagination
- Role-Based Access Control

## How to Run

1. Install XAMPP and start Apache and MySQL.
2. Copy the project folder into the `htdocs` folder.
3. Import the database into phpMyAdmin.
4. Open the project in your browser using:


http://localhost/student_auth-advanced/login.php
